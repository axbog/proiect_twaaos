Despre Mine

Salut! Numele meu este Bogdan Cristian Axinte.

Aceasta este o aplicație web dezvoltată cu Python (FastAPI) pe partea de backend și HTML/CSS/JS pe partea de frontend. Proiectul a fost implementat pentru a demonstra capacitatea de a construi un API robust, protejat prin token-uri JWT, conectat la o bază de date SQLite și publicat pe platforma Render.

Tehnologii folosite:
  • Backend: Python, FastAPI, Uvicorn, SQLite
  • Securitate: JWT, Bcrypt
  • Frontend: HTML5, CSS3, JavaScript (Vanilla), Bootstrap 5
  • Deployment: Git, Render.com
